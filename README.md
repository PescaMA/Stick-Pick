# Stick-Pick

A Foddian game about a pickaxe sticking to some blocks, bouncing off others to reach the coveted gold ore at the end.

Drag the pickaxe to swing it and reach new heights!


## Coding

Made in Rust with the Bevy game engine, this demo game is written with the ECS arhitectural pattern. 

To understand ECS and related terms, check out: [ECS-related glossary](https://github.com/SanderMertens/ecs-faq).

This project was generated using the [Bevy New 2D](https://github.com/TheBevyFlock/bevy_new_2d) template.
